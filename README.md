# kino

Это расширение для Chrome позволяющее накручивать баллы в игре "Кинопоиску 19 лет"

## Установка

Находясь в корневом каталоге выполнить:
```
yarn install
yarn build
```
Для режима разработки заменить ```yarn build``` на ```yarn start```
